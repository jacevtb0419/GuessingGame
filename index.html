Guessing Game

Terminal-based Python game — modes: Easy, Hard, Impossible

Easy

Range: 1–10

Guesses: 10

Hard

Range: 1–100

Guesses: 5

Impossible

Range: 1–100

Guesses: 3

How to run

  1. Open guessing_game.py in your Python environment.
  2. Run: python guessing_game.py
  3. Follow the terminal prompts to select a mode and enter guesses.
